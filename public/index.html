WHY IT WORKS

模型不会均匀地“随便选”。

在固定题面与采样条件下，训练数据、对齐方式和解码实现会共同塑造下一词概率。一次选择可能只是偶然；多次无上下文的独立请求，会把这种偏好沉淀为可重复测量的经验分布。

STATISTICAL NOTE

行为指纹衡量的是特定题面、参数和时间窗口下的输出分布相似性。采样温度、系统提示、供应商路由和模型更新都会改变结果；它适合作为审计线索，不应被当作模型身份的单一法律证明。

METHOD NOTE

它比较的不是答案，
而是答案背后的分布。

实验通过 OpenAI Chat Completions 或 Anthropic Messages 协议，对数字、颜色、动物、城市等固定选择题，或你定义的候选答案 Prompt 重复采样。每个样本都是一次没有历史上下文的独立请求，每个维度最终形成一个离散概率分布。

两个指纹的每个共同维度使用 Jensen–Shannon 距离比较，再取平均得到总体距离。页面同时把共同维度覆盖率、样本量和有效率折算为“采样置信度”。

距离越小，行为分布越接近；但“接近”仍可能来自模型家族、相同训练来源、路由策略或样本不足，因此结果只表达统计相似性。

DJS(P,Q) = √ ½KL(P∥M) + ½KL(Q∥M)
